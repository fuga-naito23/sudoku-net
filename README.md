# Sudoku-net
Sudoku-net is the simple Sudoku game.
We can Sudoku with friend on Online.

## description
Tech::Expertの応用カリキュラムで作成したチャットアプリです。
Ajaxの非同期通信を使用して複数ユーザでのチャットを可能にしています。
